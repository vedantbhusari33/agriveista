# 🌱 AgriVeista - Smart Farming Assistant

AgriVeista is an AI-powered web application designed to assist farmers in making better agricultural decisions. It provides intelligent crop recommendations, disease detection using images, and real-time weather insights using APIs.

---

## 🚀 Features

* 🌾 AI-based crop recommendation system
* 🌦 Real-time weather insights using API integration
* 📊 Smart farming suggestions
* 🏛 Access to government schemes and subsidies for farmers
* 🦠 AI-powered crop disease detection using images
* ⚖️ Crop comparison based on yield, cost, and suitability
* 📍 Location-based insights for better accuracy
* 📱 Mobile-friendly responsive design (works across devices)
* 🌍 Multi-language support for farmers

---

## ⚙️ How It Works

1. User enters crop or farm-related details
2. System provides crop recommendations
3. Weather data is fetched using APIs
4. User uploads crop image for disease detection
5. App suggests possible solutions and improvements
